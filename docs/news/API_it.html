OpenLabor mette a disposizione delle API RESTFul secondo lo standard internazionale Open311:

  • Scopri il servizio (discovery): XML o JSON
  • Scopri i servizi (services): XML o JSON
  • Scopri il dettaglio del servizio di ricerca di offerte di lavoro: XML o JSON
  • Scopri il dettaglio del servizio di segnalazione di offerte di lavoro: XML o JSON
  • Scopri il dettaglio del servizio di ricerca di opportunità di formazione: XML o JSON
  • Scopri il dettaglio del servizio di segnalazione di opportunità di formazione: XML o JSON
